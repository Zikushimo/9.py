# 9.py
A simple encode and decode script for a binary version of the 9dot font originaly conceived by Zikushimo.

Use:
python 9.py

This is an "A" in 9dot written in binary.

        010
    A = 101
        101

A 9dot symbol can be written as a one-line string as such.

    010
    101 = 010101101
    101
